# 📚 BookNest – Online Book Borrowing Platform

## 🌐 Live Site

https://booknest-jet.vercel.app/


## 🎯 Project Purpose

BookNest is a modern online book borrowing platform designed to digitize the traditional library system.
Users can browse books, filter by category, view details, and borrow books securely with authentication.

---

## ✨ Key Features

* 🔐 Authentication (Email & Google Login using BetterAuth)
* 📚 Browse all books with category filtering
* 🔍 Search books by title
* 📖 View detailed book information
* 📌 Private routes (only logged-in users can access)
* 📦 Borrow books with confirmation toast
* 👤 User profile management
* ✏️ Update user information (name & image)
* 🎨 Clean and responsive UI design
* ⚡ Fast performance using Next.js App Router


---

## 🛠️ Tech Stack

* Next.js (App Router)
* Tailwind CSS
* DaisyUI / HeroUI
* MongoDB
* BetterAuth
